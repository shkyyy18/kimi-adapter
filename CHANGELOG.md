# Changelog

## [0.1.0] - 2026-07-19

### Added
- Initial release of Kimi Adapter
- Local proxy that transparently forwards Claude Code requests to Kimi backend
- Automatic conversion of Claude `document` text attachments to `text` blocks
- VS Code watchdog: auto-exit when VS Code closes
- Config file support (YAML/JSON) and CLI overrides
- pip-installable package with `kimi-adapter` CLI entry point
- Docker support
- pytest test suite for document conversion
- GitHub Actions CI workflow
